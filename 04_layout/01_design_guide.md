# HUMMAN RESEARCH 홈페이지 디자인 가이드

## 디자인 기준 우선순위
1. **1순위**: Name-card.png (명함 디자인) - 색상, 로고, 브랜드 아이덴티티
2. **2순위**: activetheory.net 레이아웃 - 레이아웃 구조, 인터랙션, 모션 효과

## 브랜드 아이덴티티 (명함 디자인 기반)

### 회사명
**HUMMAN RESEARCH**

### 로고
- 원형 블루 배경
- 실버 메탈릭 기하학적 동물 머리 (영양/가젤) 모티프
- 엠보스/디보스 효과

## 컬러 시스템

### Primary Colors
- **Royal Blue**: `#1E3A8A` (또는 명함에서 추출한 정확한 색상)
  - 사용: 배경, 주요 버튼, 강조 요소
- **Metallic Silver**: `#C0C0C0` (또는 명함에서 추출한 정확한 색상)
  - 사용: 로고, 강조 텍스트, 아이콘

### Neutral Colors
- **White**: `#FFFFFF`
  - 사용: 배경, 텍스트 (블루 배경 위)
- **Black**: `#000000`
  - 사용: 주요 텍스트, 제목

### Color Usage Guidelines
- 블루는 브랜드의 핵심 색상으로 주요 섹션 배경에 사용
- 실버는 프리미엄 느낌을 주는 강조 색상으로 사용
- 흰색과 검은색은 가독성을 위한 대비 색상으로 사용

## 타이포그래피

### 폰트
- **Font Family**: Sans-serif
  - 추천: 'Inter', 'Roboto', 'Open Sans', 또는 시스템 기본 sans-serif
- **Style**: 
  - 회사명: 대문자 (UPPERCASE)
  - 본문: 일반 케이스

### Font Sizes (예시)
- H1 (메인 제목): 48px - 64px
- H2 (섹션 제목): 32px - 40px
- H3 (서브 제목): 24px - 28px
- Body (본문): 16px - 18px
- Small (캡션): 14px

## 디자인 효과

### 메탈릭 효과
- CSS를 사용한 메탈릭 텍스처 구현
- Gradient, shadow, transform을 활용한 엠보스/디보스 효과
- 예시:
  ```css
  background: linear-gradient(135deg, #E8E8E8 0%, #C0C0C0 50%, #A0A0A0 100%);
  text-shadow: 0 2px 4px rgba(0,0,0,0.3);
  ```

### 레이아웃 원칙

### 명함 디자인 기반 (1순위)
- **미니멀리즘**: 불필요한 요소 제거
- **대칭성**: 균형잡힌 레이아웃
- **계층 구조**: 명확한 정보 구조
- **여백**: 충분한 여백으로 가독성 향상

### activetheory.net 스타일 참고 (2순위)
- **대형 이미지 중심**: 전체 화면 이미지 활용
- **네거티브 스페이스**: 여백을 통한 시각적 휴식
- **슬라이딩 메뉴**: 햄버거 메뉴 + 오버레이 구조
- **히어로 섹션**: 그라디언트 조명 효과 (명함 색상 적용)

## 컴포넌트 스타일

### 버튼 (명함 디자인 기반)
- Primary: 블루 배경, 흰색 텍스트 (명함 색상)
- Secondary: 흰색 배경, 블루 텍스트, 블루 테두리
- Hover 효과: 약간의 그림자 및 색상 변화
- 메탈릭 효과 (선택적): 엠보스/디보스 스타일

### 카드/섹션
- 블루 배경 또는 흰색 배경 (명함 색상)
- 부드러운 그림자 효과
- 둥근 모서리 (border-radius: 8px - 16px)

### 네비게이션 (activetheory.net 스타일)
- 햄버거 메뉴: 왼쪽 상단 배치
- 슬라이딩 오버레이: 전체 화면 또는 대형 오버레이
- 메뉴 항목: 명함 색상 적용
- 부드러운 슬라이드 애니메이션

### 히어로 섹션 (activetheory.net + 명함)
- 명함 블루 배경 + 그라디언트 조명 효과
- 대형 비주얼 영역
- 미니멀 텍스트 배치
- 메탈릭 효과 (명함 기반)

## 반응형 디자인

- 로고: SVG 형식 권장 (확장 가능)

- 아이콘: 일관된 스타일 (선 또는 면 스타일)
