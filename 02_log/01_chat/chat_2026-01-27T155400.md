## 사용 방법

### 방법 1: Figma에서 직접 내보내기 (가장 간단)

1. Figma 링크 열기
2. 내보내고 싶은 요소 선택
3. 우측 패널에서 "Export" 클릭
4. 형식 선택 (PNG, SVG 등) 후 내보내기

### 방법 2: Python 스크립트 사용 (자동화)

1. Figma Personal Access Token 생성
   - https://www.figma.com/settings 접속
   - "Personal Access Tokens"에서 새 토큰 생성

2. `.env` 파일 생성
   ```bash
   # .env.example을 복사하여 .env 파일 생성
   # .env 파일에 토큰 추가
   FIGMA_ACCESS_TOKEN=your_token_here
   ```

3. 필요한 패키지 설치
   ```bash
   venv\Scripts\activate
   pip install requests
   ```

4. 스크립트 실행
   ```bash
   python 03_script/07_export_figma_assets.py
   ```

스크립트는 제공하신 Figma URL에서 자동으로 파일 키와 노드 ID를 추출하여 이미지를 내보냅니다.

### 방법 3: Dev Mode 사용 (코드 추출)

1. Figma에서 "Dev Mode" 토글 활성화
2. 요소 선택 시 자동으로 CSS/코드 표시
3. 코드 복사하여 프로젝트에 적용
