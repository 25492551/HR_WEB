## 수행 작업

### 1. 로고 이미지 SVG 생성

#### logo.svg (일반 로고)
- 크기: 200x200px
- 원형 블루 배경 (#1E3A8A)
- 실버 메탈릭 영양/가젤 머리 모티프
- 엠보스/디보스 효과
- 메탈릭 그라디언트 적용
- 하이라이트 효과

#### logo-circle.svg (원형 로고, 히어로 섹션용)
- 크기: 200x200px
- 원형 블루 배경 (라디얼 그라디언트 적용)
- 실버 메탈릭 영양/가젤 머리 모티프
- 향상된 엠보스 효과
- 더 강한 하이라이트 효과

#### favicon.svg (파비콘)
- 크기: 64x64px
- 원형 블루 배경
- 단순화된 영양/가젤 머리 모티프
- 작은 크기에 최적화된 디자인

### 2. 로고 디자인 특징

#### 명함 디자인 기반 요소
- **원형 블루 배경**: Royal/Cobalt Blue (#1E3A8A)
- **실버 메탈릭 효과**: 
  - 그라디언트: #E8E8E8 → #C0C0C0 → #A0A0A0
  - 엠보스/디보스 필터 효과
  - 하이라이트 오버레이
- **기하학적 동물 머리**: 
  - Low-poly 스타일
  - 영양/가젤 머리 모티프
  - 긴 곡선 뿔
  - 정면을 향한 얼굴

#### SVG 기술적 특징
- 벡터 기반 (확대/축소 시 품질 손실 없음)
- CSS에서 크기 조절 가능
- 메탈릭 효과가 SVG 필터로 구현
- 접근성 고려 (의미 있는 구조)

### 3. 이미지 최적화 가이드 작성

#### 문서 작성
- `06_docs/02_image_optimization_guide.md` 생성
- 이미지 최적화 방법 상세 안내

#### 가이드 내용
1. **로고 이미지**: SVG 형식 사용 가이드
2. **서비스 섹션 이미지**: 
   - 권장 크기: 1920x1080px
   - JPEG 품질: 80-85%
   - 파일 크기: 200KB 이하
   - Lazy loading 적용
3. **이미지 형식 선택**: JPEG, WebP, PNG, SVG 가이드
4. **최적화 도구**: 온라인/명령줄 도구 안내
5. **반응형 이미지**: srcset, picture 요소 사용법
6. **SEO 이미지**: Open Graph, Twitter Card 이미지 가이드
7. **성능 최적화 팁**: Lazy loading, 캐싱, CDN 등

### 4. 아이콘 준비

#### 로고 내 아이콘 요소
- 영양/가젤 머리 모티프가 아이콘 역할
- 메탈릭 효과 적용
- 다양한 크기로 사용 가능 (SVG 벡터)

#### 추가 아이콘 (필요시)
- About 섹션 카드 아이콘은 CSS로 구현 가능
- 또는 별도 SVG 아이콘 추가 가능
