### 4. 시맨틱 HTML 적용

#### 시맨틱 태그 사용
- `<header>`: 헤더 영역
- `<nav>`: 네비게이션
- `<main>`: 메인 콘텐츠
- `<section>`: 섹션 구분
- `<article>`: 독립적인 콘텐츠 (서비스 아이템)
- `<footer>`: 푸터 영역
- `<address>`: 연락처 주소

#### ARIA 속성
- `role` 속성: banner, navigation, main, contentinfo
- `aria-label`: 접근성을 위한 레이블
- `aria-expanded`: 메뉴 상태
- `aria-controls`: 제어 대상
- `aria-hidden`: 시각적으로 숨김 요소
- `aria-required`: 필수 입력 필드

#### 접근성 고려
- Skip to main content 링크
- 적절한 heading 계층 구조
- 이미지 alt 텍스트
- 폼 label 연결
- 키보드 네비게이션 고려
