### 2. 정의된 변수 카테고리

#### 색상 시스템
- Primary Colors: Royal Blue (#1E3A8A), Metallic Silver (#C0C0C0)
- Neutral Colors: White (#FFFFFF), Black (#000000)
- Color Variations: Light/Dark variations, Gray scale
- Background Colors: Primary, Secondary, Overlay
- Text Colors: Primary, Secondary, Accent, Muted

#### 타이포그래피
- Font Families: Primary (Inter, Roboto, Open Sans), Mono
- Font Sizes: H1-H3, Body, Small, Caption (clamp() 사용으로 반응형)
- Font Weights: Light, Normal, Medium, Semibold, Bold
- Line Heights: Tight, Normal, Relaxed
- Letter Spacing: Tight, Normal, Wide, Wider (대문자용)

#### 스페이싱 시스템
- 8단계 스페이싱: xs(4px) ~ 4xl(96px)

#### Border & Radius
- Border Radius: sm(4px) ~ full(9999px)
- Border Width: Thin, Medium, Thick

#### Shadows
- 기본 Shadow: sm ~ 2xl
- 메탈릭 효과 Shadow: Metallic, Emboss, Deboss

#### 메탈릭 효과 (명함 디자인 기반)
- Metallic Gradient
- Metallic Text Shadow
- Hero Section Gradient (activetheory.net 스타일 + 명함 색상)
- Gradient Lighting Effect

#### 전환 및 애니메이션
- Transition: Fast, Base, Slow, Slower
- Animation Duration: Fast ~ Slower

#### Breakpoints
- Mobile: 320px - 767px
- Tablet: 768px - 1023px
- Desktop: 1024px+
- Desktop Large: 1280px+
- Desktop XL: 1920px+

#### Z-Index 레이어
- Base ~ Tooltip
- Menu Overlay: 800 (activetheory.net 스타일 슬라이딩 메뉴용)

#### 레이아웃
- Container Max Width: 1280px
- Container Padding: Mobile, Tablet, Desktop
- Section Padding: Mobile, Tablet, Desktop

#### 컴포넌트별 변수
- Navigation: 햄버거 메뉴 크기, 오버레이 배경, 전환 효과
- Buttons: 패딩, 보더 반경, 전환 효과
- Cards/Sections: 보더 반경, 패딩, 그림자
- Hero Section: 최소 높이 (Mobile, Tablet, Desktop)
