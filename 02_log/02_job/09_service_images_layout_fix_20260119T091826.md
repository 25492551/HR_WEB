# 작업 로그: 서비스 이미지 “저품질(겹침/잘림)” 인상 개선

## 작업 일시
- 2026-01-19T091826

## 배경 / 문제
- `public/images/service-1~3.jpg`가 차트/그래프 형태(내부 텍스트, 격자, 작은 라벨 포함)라서, 서비스 섹션에서 **풀스크린 배경(cover)** 으로 사용 시 텍스트/오브젝트가 잘리고 페이지의 서비스 타이틀과 **시각적으로 충돌(겹침/중복)** 하며 “저품질”처럼 보이는 문제가 발생함.

## 변경 내용
- **서비스 섹션 HTML 구조 보강**
  - 서비스 아이템에 서비스별 클래스 추가: `service-item--research`, `service-item--consulting`, `service-item--analysis`
  - 서비스 텍스트 영역에 카드 래퍼 추가: `.service-content-inner`
  - 파일: `public/index.html`
- **서비스 배경 이미지의 시각적 잡음 완화**
  - 배경 이미지에 약한 블러/톤 조정 적용(텍스트/격자 디테일을 의도적으로 “배경화”)
  - 서비스별 `object-position` 미세 조정으로 내부 텍스트/가장자리 요소가 과도하게 보이거나 잘리는 인상을 완화
  - 오버레이를 브랜드 블루 기반 그라디언트+라이트 효과로 교체하여 색조 일관성 강화
  - 파일: `public/css/style.css`

## 기대 효과
- 이미지 내부 텍스트와 페이지 서비스 타이틀의 **중복/겹침 인상 감소**
- 배경 디테일(격자, 작은 글자)이 전면에 튀지 않아 **가독성 및 프리미엄 톤 개선**
- 오버레이 색감이 브랜드 컬러와 맞아 **일관성 향상**
