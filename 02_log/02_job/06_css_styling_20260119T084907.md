## 수행 작업

### 1. Reset/Normalize CSS 적용
- 기본 스타일 리셋
- Box-sizing: border-box
- 기본 마진/패딩 제거
- 리스트, 링크, 버튼, 입력 필드 기본 스타일 제거
- 이미지 기본 스타일 설정
- 폰트 스무딩 적용

### 2. 레이아웃 시스템 구축

#### Grid 레이아웃
- About Section: 1열 (모바일) → 2열 (태블릿+)
- Contact Section: 1열 (모바일) → 2열 (태블릿+)
- Footer: 1열 (모바일) → 2열 (태블릿+)

#### Flexbox 레이아웃
- Navigation: flex로 정렬
- Hero Section: flex로 중앙 정렬
- 컴포넌트 내부 정렬

### 3. 반응형 디자인 구현

#### Mobile First 접근
- 기본 스타일은 모바일 기준
- 미디어 쿼리로 태블릿/데스크톱 확장

#### Breakpoints
- Tablet: 768px
- Desktop: 1024px
- Large Desktop: 1280px

#### 반응형 타이포그래피
- clamp() 함수 사용
- H1: clamp(2.5rem, 5vw, 4rem)
- H2: clamp(2rem, 4vw, 2.5rem)
- H3: clamp(1.5rem, 3vw, 1.75rem)

### 4. 섹션별 스타일링

#### Header/Navigation (activetheory.net 스타일)
- **햄버거 메뉴**
  - 3줄 아이콘 구조
  - 클릭 시 X 모양으로 변환 애니메이션
  - 명함 블루 색상 적용
- **슬라이딩 오버레이 메뉴**
  - 전체 화면 오버레이
  - 어두운 배경 (rgba(0, 0, 0, 0.95))
  - 중앙 정렬 메뉴 항목
  - 메뉴 링크 호버 효과 (밑줄 애니메이션)
  - 명함 색상 적용
- **로고**
  - 중앙 배치
  - 명함 블루 색상

#### Hero Section (activetheory.net + 명함)
- **배경 레이어**
  - 그라디언트 배경 (명함 블루)
  - 조명 효과 레이어 (radial-gradient)
- **콘텐츠**
  - 중앙 정렬
  - 원형 로고 (메탈릭 효과)
  - 대형 타이틀 (메탈릭 텍스트 섀도우)
  - CTA 버튼
- **스크롤 인디케이터**
  - 바운스 애니메이션
  - 화살표 아이콘

#### About Section (명함 디자인 우선)
- 섹션 헤더 (중앙 정렬)
- 그리드 레이아웃 (텍스트 + 카드)
- 카드 스타일
  - 메탈릭 효과 아이콘
  - 호버 효과 (위로 이동, 그림자 증가)
  - 명함 색상 적용

#### Services Section (activetheory.net 스타일)
- **전체 화면 이미지**
  - 각 서비스 아이템: 100vh 높이
  - 이미지: object-fit: cover
  - 오버레이 그라디언트
- **콘텐츠**
  - 중앙 정렬
  - 대형 타이틀
  - 미니멀 설명 텍스트
  - 흰색 텍스트 (이미지 위)

#### Contact Section (명함 디자인 우선)
- 섹션 헤더
- 그리드 레이아웃 (연락처 정보 + 폼)
- 연락처 정보 카드
  - 명함 색상 강조
  - 카드 스타일
- 연락처 폼
  - 입력 필드 스타일
  - Focus 상태 스타일
  - 버튼 스타일

#### Footer (명함 디자인 우선)
- 블루 배경 (명함 색상)
- 흰색 텍스트
- 그리드 레이아웃 (브랜드 정보 + 저작권/네비게이션)
- 로고 및 회사명
- 푸터 네비게이션

### 5. 컴포넌트 스타일

#### 버튼 (명함 디자인 기반)
- **Primary Button**
  - 블루 배경 (명함 색상)
  - 흰색 텍스트
  - 그림자 효과
  - 호버: 어두운 블루, 그림자 증가, 위로 이동
- **Secondary Button**
  - 투명 배경
  - 블루 테두리 및 텍스트
  - 호버: 블루 배경, 흰색 텍스트

#### 카드
- 흰색 배경
- 둥근 모서리
- 그림자 효과
- 호버: 위로 이동, 그림자 증가

#### 로고
- 메탈릭 효과 (drop-shadow)
- 반응형 크기

### 6. 유틸리티 클래스
- `.container`: 최대 너비, 패딩
- `.section-header`: 섹션 헤더 스타일
- `.section-title`: 섹션 제목
- `.section-subtitle`: 섹션 서브타이틀
- `.skip-link`: 접근성 스킵 링크

### 7. 접근성 고려

#### Focus 스타일
- 모든 인터랙티브 요소에 focus outline
- 명함 블루 색상 outline

#### Reduced Motion
- `prefers-reduced-motion` 미디어 쿼리 지원
- 애니메이션 및 전환 효과 비활성화

#### Skip Link
- 키보드 사용자를 위한 스킵 링크
- 포커스 시 표시
