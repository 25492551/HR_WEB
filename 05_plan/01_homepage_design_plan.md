# 홈페이지 디자인 계획

## 프로젝트 개요
- **회사명**: HUMMAN RESEARCH
- **프로젝트명**: 웹 홈페이지 제작
- **디자인 기준**: 
  1. **1순위**: Name-card.png (명함 디자인) - 주요 디자인 기준
  2. **2순위**: activetheory.net 레이아웃 - 레이아웃 및 인터랙션 참고

## 명함 디자인 분석

### 색상 팔레트
- **주요 블루**: 로얄/코발트 블루 (Royal/Cobalt Blue)
  - 배경색으로 사용
  - 브랜드 아이덴티티의 핵심 색상
- **흰색**: 메인 배경 및 대비 색상
- **실버/메탈릭 그레이**: 
  - 로고의 동물 머리 모티프
  - 텍스트 강조 (예: "SeungUn Lee")
  - 메탈릭, 엠보스/디보스 효과
- **검은색**: 
  - 주요 텍스트 색상 (예: "HUMAN RESEARCH")
  - 가독성을 위한 대비 색상

### 디자인 요소
1. **원형 로고**
   - 블루 원형 배경
   - 실버 메탈릭 기하학적 동물 머리 (영양/가젤)
   - 엠보스/디보스 효과

2. **타이포그래피**
   - 산세리프 폰트
   - 대문자 사용 ("HUMAN RESEARCH")
   - 메탈릭 효과가 적용된 텍스트

3. **레이아웃**
   - 깔끔하고 미니멀한 디자인
   - 대칭적이고 균형잡힌 구성
   - 명확한 계층 구조

## 홈페이지 디자인 방향성

### 컬러 스킴
- **Primary Color**: 로얄/코발트 블루 (#1E3A8A 또는 유사한 색상)
- **Secondary Color**: 실버/메탈릭 그레이 (#C0C0C0 또는 유사한 색상)
- **Background**: 흰색 (#FFFFFF)
- **Text**: 검은색 (#000000) 및 흰색 (블루 배경 위)

### 디자인 컨셉
1. **프리미엄 & 프로페셔널**
   - 메탈릭 효과와 블루 색상으로 고급스러운 느낌
   - 깔끔하고 전문적인 레이아웃

2. **미니멀리즘**
   - 불필요한 요소 제거
   - 명확한 정보 전달

3. **브랜드 일관성**
   - 명함 디자인과 동일한 색상 및 스타일
   - 로고와 브랜드 요소의 일관된 사용

## activetheory.net 레이아웃 참고 요소

### 주요 특징 (2순위 참고)
1. **인터랙티브 효과**
   - 마우스 따라 움직이는 시각 효과 (Fractured glass effect)
   - WebGL/Shader 기반 고급 시각 효과 (선택적 적용)
   - 마우스 움직임에 반응하는 미세한 파라랙스 효과

2. **네비게이션**
   - 햄버거 메뉴 (왼쪽 상단)
   - 슬라이딩 메뉴 오버레이 (전체 화면 또는 대형 오버레이)
   - 메뉴 항목과 로고 중심 배치

3. **히어로 섹션**
   - 어두운 배경 + 그라디언트 조명 효과 (명함 블루 색상 적용)
   - 대형 이미지 중심 콘텐츠
   - 미니멀 텍스트 배치

4. **콘텐츠 레이아웃**
   - 전체 화면 이미지 사용
   - 텍스트 최소화 및 하단 간단한 설명
   - 이미지 중심의 시각적 스토리텔링

5. **타이포그래피**
   - 산세리프 폰트 (명함과 일치)
   - 평평하고 모던한 조합
   - 깨끗하고 미니멀한 스타일

6. **모션 및 애니메이션**
   - 부드러운 전환 효과
   - 호버 시 미세한 움직임
   - 스크롤 기반 파라랙스 효과

## 구현 계획

### 1단계: 기본 구조 설계
- HTML 구조 설계 (activetheory.net 스타일 레이아웃 참고)
- CSS 변수 정의 (명함 색상 팔레트 우선)
- 반응형 레이아웃 계획

### 2단계: 디자인 시스템 구축
- 로고 디자인/적용 (명함 디자인 기반)
- 타이포그래피 시스템 (명함 산세리프 + activetheory.net 모던 스타일)
- 버튼 및 UI 컴포넌트 스타일 (명함 색상 적용)

### 3단계: 섹션별 디자인
- **헤더/네비게이션** (activetheory.net 스타일)
  - 햄버거 메뉴 (왼쪽 상단)
  - 슬라이딩 오버레이 메뉴
  - 명함 색상 적용
- **히어로 섹션** (activetheory.net 스타일 + 명함 색상)
  - 명함 블루 배경 + 그라디언트 조명 효과
  - 대형 이미지 또는 비주얼
  - 미니멀 텍스트
- **회사 소개** (명함 디자인 우선)
  - 명함 색상 및 스타일 반영
- **서비스/제품 소개** (activetheory.net 스타일)
  - 전체 화면 이미지 중심
  - 최소한의 텍스트 설명
- **연락처/푸터** (명함 디자인 우선)

### 4단계: 인터랙션 및 애니메이션
- **명함 기반 효과** (1순위)
  - 메탈릭 엠보스/디보스 효과 (CSS)
  - 명함 색상 기반 그라데이션
- **activetheory.net 스타일 효과** (2순위)
  - 마우스 따라 움직이는 효과 (선택적, WebGL 또는 CSS 기반)
  - 부드러운 스크롤 효과
  - 호버 시 미세한 움직임
  - 슬라이딩 메뉴 애니메이션

## 기술 스택 고려사항

### 필수 기술
- HTML5, CSS3
- JavaScript (Vanilla JS)
- 반응형 디자인 (모바일, 태블릿, 데스크톱)

### 효과 구현
- **명함 기반 효과** (1순위)
  - 메탈릭 효과: CSS gradients, shadows, transforms
  - 엠보스/디보스 효과: CSS 3D transforms, box-shadow
- **activetheory.net 스타일 효과** (2순위, 선택적)
  - 인터랙티브 효과: JavaScript + CSS animations
  - 고급 시각 효과: WebGL/Three.js (선택적, 성능 고려)
  - 파라랙스 효과: JavaScript scroll events + CSS transforms

### 성능 최적화
- WebGL 효과는 선택적 적용 (성능 영향 고려)
- CSS 기반 애니메이션 우선 사용
- 이미지 최적화 및 lazy loading
